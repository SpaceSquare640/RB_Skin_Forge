RB Skin Forge - Windows Installer
==================================

To install:  double-click  RB_Skin_Forge_installer.exe

What it does:
  - Installs the Windows (.exe) version of RB Skin Forge.
  - Default location: C:\Program Files\RB Skin Forge  (you can change it during setup).
  - Lets you choose whether to create a desktop shortcut.
  - Adds a Start menu entry and an entry in "Apps & features" for clean uninstall.

Notes:
  - Installing to Program Files asks for administrator permission (standard Windows behaviour).
  - This is a self-contained build - no .NET runtime or other installs are required.

Files in this folder:
  RB_Skin_Forge_installer.exe  - the installer (share/distribute this single file)
  RB_Skin_Forge.iss            - the Inno Setup build script (source; rebuild with ISCC.exe)
