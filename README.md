# RB Skin Forge — Windows Package (portable)

This branch holds the portable Windows build of **RB Skin Forge** (latest: v0.2.0),
packaged as `Package_Program_EXE.zip`. It is self-contained — no .NET runtime or
other installs are required.

## Run
1. Download and unzip `Package_Program_EXE.zip`.
2. Open the extracted `Package_Program_EXE` folder and run `RB_Skin_Forge.exe`.

Prefer a guided installer (Start-menu entry, desktop shortcut, uninstaller)?
Download `RB_Skin_Forge_installer.exe` from the [Releases page](../../releases).
