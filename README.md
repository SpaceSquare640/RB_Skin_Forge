# RB Skin Forge — Android Package

This branch holds the Android build of **RB Skin Forge** (latest: v0.2.0).

## Install
Download `Package_Program_APK/RB_Skin_Forge.apk` and open it on your Android
device. You may need to allow "Install unknown apps" for your browser or file
manager the first time. A ready-made copy is also on the
[Releases page](../../releases).
