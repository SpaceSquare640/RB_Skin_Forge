# RB Skin Forge — HTML (Web) Package

This branch holds the web build of **RB Skin Forge** (Blazor WebAssembly, latest: v0.2.0).

## Run
Download the `HTML_Version` folder, then double-click
`HTML_Version/RB_Skin_Forge_launcher.bat`. It starts a small local web server
(no admin rights or installs needed) and opens the app in your browser.

Note: the app must be served this way — opening `app/index.html` directly from
disk will not work. A ready-made `HTML_Version.zip` is on the
[Releases page](../../releases), or try the hosted demo:
https://spacesquare640.github.io/RB_Skin_Forge/
